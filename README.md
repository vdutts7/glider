# Glider

Chrome extension that bridges browser tabs to local development tools via Chrome DevTools Protocol (CDP).

## Install

1. Install from [Chrome Web Store](#) (pending)
2. Or load unpacked from this repo

## Usage

1. Start local relay server on port 19988
2. Click extension icon on any tab
3. Tab is now connected to your local tools
